## MÓDULOS DEL SISTEMA (MVP)

### 1. SCOUT AGENT
- Busca oportunidades: festivales, showcases, open calls, playlists, blogs
- Foco en escena de Bogotá y LATAM
- Clasifica por: prioridad, compatibilidad EOS, deadline, valor estratégico

### 2. OUTREACH AGENT
- Redacta emails profesionales, propuestas, follow-ups
- Organiza contactos y oportunidades de networking
- **Regla:** Aprobación humana siempre antes de enviar

### 3. TREND ANALYZER
- Monitorea TikTok, Instagram, YouTube Shorts
- Detecta: hooks de alto rendimiento, tendencias cinematográficas, comportamiento de audiencia
- Objetivo: inteligencia cultural, NO copiar tendencias

### 4. CREATIVE AGENT
- Ideas cinematográficas, captions, hooks, conceptos documentales
- Prompts para IA visual, estructuras de contenido
- Todo alineado con identidad EOS

### 5. MEMORY SYSTEM
- Memoria persistente de decisiones creativas, branding, campañas, referencias
- Hace el sistema coherente, vivo y contextual

### 6. COMPETITIVE INTELLIGENCE
- Analiza artistas emergentes, cinematográficos, emotivos
- Detecta: estéticas saturadas, gaps de diferenciación, errores de otros
- Posiciona EOS de forma única
